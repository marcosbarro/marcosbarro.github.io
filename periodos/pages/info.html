Al ejecutar el TEST, se muestran tres bloques con periodos valor:

  • El bloque superior contiene una lista de periodos-valor generados aleatoriamente. El valor es el numero mostrado en cada periodo
  • En el bloque central se nuestra la lista de periodos-valor normalizada equivalente. Los valores de periodos solapados simplemente se suman.
  • El bloque inferior es solo informativo para mostrar la lista de periodos-valor normalizada partida en cada hora del dia. Realmente no es una lista normalizada (puede tener periodos contiguos con el mismo valor), sino un ejemplo de como se puede hacer uso de una lista normalizada.
