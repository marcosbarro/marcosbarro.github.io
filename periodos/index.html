Listas de Periodo-Valor Normalizadas

Al ejecutar el TEST, se generan tres bloques con periodos-valor con inicio desde las cero horas del dia actual hasta las cero horas del dia siguiente:

  • El bloque superior contiene una lista de 10 periodos-valor generados aleatoriamente. El valor es el numero mostrado en cada periodo
  • En el bloque central se nuestra la lista de periodos-valor normalizada equivalente (los valores de periodos solapados simplemente se suman)
  • El bloque inferior es solo informativo para mostrar la lista de periodos-valor normalizada partida en las 24 horas del dia. Realmente no es una lista normalizada (puede tener periodos contiguos con el mismo valor), sino un ejemplo de como se puede hacer uso de una lista normalizada.
